# App-Notifier

App-Notifier is a simple tool that shows popups about running applications and lets you quickly manage them.

## How to use

1. Download the `.exe` file
2. Run the `.exe` file
3. The app will start running in the background

**Tip:** You can place it in the startup folder so it starts automatically when you log in.

## Popup functions

When a popup appears, you will see 3 buttons:

* **✕** — Closes the popup
* **🚫** — Closes the opened app
* **🔍** — Searches the app online

## Customization / Closing the app

You can find the app icon in the system tray (sometimes hidden under the ⌃ icon).

![App tray](https://github.com/user-attachments/assets/c72a597c-d030-4b50-8124-083899afaef4)

Right-click the icon to open the menu. You can:

* Toggle sound on/off
* Change popup theme (red / yellow / green / blue / purple)
* Toggle extensions (.exe, .pdf etc.)
* Quit the app
